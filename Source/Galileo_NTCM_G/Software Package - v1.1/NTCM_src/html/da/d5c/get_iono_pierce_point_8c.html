FUNCTION

This module computes the geographic latitude and longitude of the Ionospheric Pierce Point (IPP) (Described in Section 2.5.2). IPP is the point where the Line of Sight (LoS) intersects the reference ionospheric layer, this latter set at an altitude of 450Km.

Parameters
[in]llhUser=> Receiver position in Latitude[rad], Longitude[rad], height[m]
[in]ElRad=> Elevation (DoA) [rad]
[in]AzRad=> Azimuth (DoA) [rad]
[out]IppCoord=>Pierce Point coordinates in Latitude[rad], Longitude[rad]
